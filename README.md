# Python Interview Prep

This repository is for Python coding interview preparation focused on Data Engineer / Azure Data Engineer roles.

## Goal

The goal is to become confident in solving easy and medium Python interview problems using clean logic, readable code, and proper time complexity explanation.

## Topics

1. Python Basics
2. Lists / Arrays
3. Strings
4. Dictionaries / Hash Maps
5. Sets
6. Sorting
7. Two Pointers
8. Sliding Window
9. Stack
10. Queue
11. Recursion Basics
12. Time Complexity
13. Mixed Problems

## Practice Format

Each problem will include:

- Problem statement
- Input and output example
- Approach
- Python solution
- Time complexity
- Space complexity
- Mistake notes if needed

## Target

- Easy problems: strong
- Medium problems: comfortable
- Hard problems: optional later

## Progress

This repo will be updated module by module.
